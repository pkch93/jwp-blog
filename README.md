# 우테코 Level2 미션 myblog

> 나만의 블로그 서비스 만들기

## 기능

### 인증
- 로그인
- 로그아웃

### 회원 관련
- 회원가입
- 마이페이지 정보 확인
- 내 정보 수정
- 회원 탈퇴

### 글쓰기 (Article)
- 글 작성
- 글 조회
- 글 수정
- 글 삭제

### 댓글 (Comment)
- 댓글 작성
- 댓글 조회
- 댓글 수정
- 댓글 삭제

## 고려사항
1. Session 저장에 `User` 저장 정보 변경에 따른 검증 로직 변경
2. ArgumentResolver로 파라미터에서 Session에 저장된 User 받아오기
